ARKitQR

Overview

ARKitQR is an iOS application developed for collecting localization, sensor, and visual feature data from Apple’s ARKit.

The application records position estimation results, acceleration data, and visual feature information generated by ARKit, enabling the analysis of Visual-Inertial Odometry (VIO) behavior in mobile Augmented Reality (AR) environments.

This system was developed as part of research investigating localization errors in mobile AR systems.

Development Environment

Hardware

* iPhone 17 Pro

Software

* iOS
* Swift
* ARKit
* Xcode

Features

* ARKit-based environment initialization
* Real-time localization tracking
* Position and acceleration logging
* Visual feature point logging
* Automatic experiment timing
* Automatic email-based CSV export

Recorded Data

ARKitData.csv

Records localization and sensor information generated by ARKit.

Field	Description
Timestamp	Timestamp of the measurement
Distance	Total displacement from the initial position
DirectionX	Displacement along the ARKit X-axis
DirectionY	Displacement along the ARKit Y-axis
DirectionZ	Displacement along the ARKit Z-axis
AccelerationX	Acceleration along the X-axis
AccelerationY	Acceleration along the Y-axis
AccelerationZ	Acceleration along the Z-axis
Features	Number of visual feature points detected in the current frame

ARKitFeaturePoints.csv

Records detailed information for each visual feature point detected by ARKit.

Field	Description
Timestamp	Timestamp of the measurement
FrameIndex	Frame number
FeatureIndex	Index of the feature point within the frame
FeatureID	Unique identifier assigned to the feature point
X	X coordinate of the feature point in 3D space
Y	Y coordinate of the feature point in 3D space
Z	Z coordinate of the feature point in 3D space
ScreenX	X coordinate of the feature point on the screen
ScreenY	Y coordinate of the feature point on the screen
FeatureCount	Total number of visual feature points detected in the corresponding frame

Setup

1. Configure the destination email address in ARKitQR/MailHandler.swift.
2. Build and install the application using Xcode.
3. Launch the application on an iPhone.

Usage

1. Launch the application.
2. Move the device and allow ARKit to scan the surrounding environment.
3. When ARKit initialization is completed, the Start button turns blue.
4. Press the Start button to begin data collection.
5. Localization, acceleration, and visual feature data are recorded automatically.
6. After the predefined measurement period (210 seconds), the email composition screen is launched automatically.
7. The recorded CSV files can then be sent via email.

Research Applications

This application is designed for evaluating localization performance in mobile AR systems and analyzing the behavior of ARKit-based Visual-Inertial Odometry.

The collected data can be used to:

* Analyze localization errors
* Evaluate visual feature point behavior
* Investigate the relationship between visual and inertial information
* Analyze ARKit performance under different environmental conditions
* Support research on localization robustness in mobile AR systems
